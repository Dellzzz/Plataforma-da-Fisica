01

O QUE SÃO VETORES?

Um vetor é uma grandeza que possui módulo (intensidade), direção (reta de ação) e sentido (para onde aponta). São representados por setas. Diferem das grandezas escalares, que têm apenas módulo.

Vetores aparecem em toda a Física: força, velocidade, aceleração, campo elétrico, campo magnético. Dominar vetores é essencial para resolver problemas de lançamentos, dinâmica e eletromagnetismo.

ESCALARES × VETORIAIS

ESCALARES: apenas número + unidade
Ex: massa (5 kg), temperatura (20°C), energia (100 J), tempo (3 s)

VETORIAIS: módulo + direção + sentido
Ex: velocidade (5 m/s para leste), força (10 N para cima)

REPRESENTAÇÃO

✦ Seta: comprimento = módulo, direção e sentido pelo apontamento
✦ Notação: ⃗A, A, ou A com seta
✦ Componentes: A = (A_x, A_y) ou A = A_x î + A_y ĵ
✦ Módulo: |A| = √(A_x² + A_y²)

VETORES NOTÁVEIS

✦ Vetor nulo: módulo = 0 (ponto)
✦ Vetores unitários: |A| = 1 (î, ĵ, k̂)
✦ Vetores opostos: mesmo módulo e direção, sentidos opostos
✦ Vetores colineares: mesma direção

COMPONENTES RETANGULARES

A_x = |A|·cosθ   |   A_y = |A|·senθ   |   |A| = √(A_x² + A_y²)
θ = ângulo com eixo x positivo  |  θ = arctan(A_y / A_x) (ajustado para o quadrante)

DECOMPOSIÇÃO

Qualquer vetor pode ser decomposto em duas componentes perpendiculares (horizontal e vertical). Isso transforma problemas vetoriais em dois problemas escalares independentes.

A = A_x · î + A_y · ĵ
A_x = |A|·cosθ (componente x)
A_y = |A|·senθ (componente y)

MÓDULO E ÂNGULO

Dadas as componentes, recuperamos o módulo e a direção original:

|A| = √(A_x² + A_y²)
θ = arctan(A_y / A_x)

Atenção ao quadrante!
arctan retorna valor entre −90° e 90°

VETORES UNITÁRIOS î E ĵ

✦ î = vetor unitário na direção x (+)
✦ ĵ = vetor unitário na direção y (+)
✦ |î| = |ĵ| = 1
✦ î ⊥ ĵ (perpendiculares)

A = 3î + 4ĵ → |A| = √(9+16) = 5
θ = arctan(4/3) ≈ 53°

ADIÇÃO DE VETORES

MÉTODOS DE ADIÇÃO

Vetores são somados de dois modos: graficamente (regra do triângulo/paralelogramo) ou analiticamente (soma das componentes — mais preciso).

R⃗ = A⃗ + B⃗  |  R_x = A_x + B_x  |  R_y = A_y + B_y
|R| = √(R_x² + R_y²)  |  θ_R = arctan(R_y/R_x)

REGRA DO TRIÂNGULO

Posicione a origem de B⃗ na ponta de A⃗. O vetor resultante R⃗ vai da origem de A⃗ à ponta de B⃗.

1. Desenhe A⃗
2. Na ponta de A⃗, desenhe B⃗
3. R⃗ = da origem de A⃗ à ponta de B⃗

MÉTODO ANALÍTICO

Decompõe cada vetor em componentes e soma separadamente. Funciona para qualquer número de vetores.

R_x = A_x + B_x + ...
R_y = A_y + B_y + ...
|R| = √(R_x² + R_y²)

SUBTRAÇÃO E MULTIPLICAÇÃO ESCALAR

SUBTRAÇÃO DE VETORES

A⃗ − B⃗ = A⃗ + (−B⃗). O vetor −B⃗ tem o mesmo módulo e direção de B⃗, mas sentido oposto.

D⃗ = A⃗ − B⃗ = A⃗ + (−B⃗)
D_x = A_x − B_x
D_y = A_y − B_y

MULTIPLICAÇÃO POR ESCALAR

Multiplica cada componente pelo escalar k.

k·A⃗ = (k·A_x, k·A_y)
k > 0: mesmo sentido
k < 0: sentido oposto
|k·A⃗| = |k|·|A⃗|

VETOR DIFERENÇA (Δv⃗)

Em cinemática, a variação de velocidade é um vetor:

Δv⃗ = v⃗_f − v⃗_i
a⃗ = Δv⃗ / Δt

Importante: mesmo que |v_f| = |v_i|,
se direção mudou, Δv⃗ ≠ 0

PRODUTO ESCALAR

DEFINIÇÃO

O produto escalar (produto interno) entre dois vetores A⃗ e B⃗ resulta em um escalar. Mede o quanto os vetores estão "alinhados".

A⃗·B⃗ = |A|·|B|·cosθ = A_x·B_x + A_y·B_y
θ = ângulo entre os vetores  |  Resultado é um ESCALAR (número)

INTERPRETAÇÃO GEOMÉTRICA

O produto escalar é a projeção de A⃗ sobre B⃗ multiplicada por |B⃗| (ou vice-versa). Usado para calcular trabalho: W = F⃗·d⃗ = |F||d|cosθ.

A⃗·B⃗ = 0 → vetores perpendiculares
A⃗·B⃗ > 0 → ângulo agudo (<90°)
A⃗·B⃗ < 0 → ângulo obtuso (>90°)
A⃗·A⃗ = |A|² (produto por si mesmo)
